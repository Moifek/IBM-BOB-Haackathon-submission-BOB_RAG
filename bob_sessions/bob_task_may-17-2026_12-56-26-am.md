295 | | Pitfall                                                       | Mitigation                                                  |
296 | |---------------------------------------------------------------|--------------------------------------------------------------|
297 | | `langchain-ollama` version drift breaks `embed_documents`     | Pin to `langchain-ollama>=0.2.0,<0.4.0` in `pyproject.toml`  |
298 | | Chroma defaults to a telemetry call on first init             | Always pass `Settings(anonymized_telemetry=False)`          |

300 | | Embedding dim mismatch between query and stored vecs          | Use the **same** `OLLAMA_EMBED_MODEL` for ingest and search |
301 | | Tests accidentally touch real Ollama                          | Mock `langchain_ollama.OllamaEmbeddings` and `ChatOllama` at the import site |
